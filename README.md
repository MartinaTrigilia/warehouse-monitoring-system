# Inventory Order Management

This project is a Python-based warehouse and order management system developed for a fictional B2B e-commerce company. It allows users to manage inventory, process customer orders, monitor stock levels, generate daily reports, and store data using CSV files.

## Features

- Product catalog management
- Order registration and fulfillment
- Stock availability checks
- Reorder point monitoring
- Daily reporting
- CSV data persistence
- Activity logging
- Command-line interface
